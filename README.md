## Air Ticket Reservation System ‚Äì Python

A simple **menu-driven Air Ticket Reservation System** built using Python.
The program allows users to register customer details, select travel class, calculate ticket prices, order food, compute luggage charges, and generate a complete final bill.

## Features

### **1. Customer Registration**

* Stores passenger name, address, journey date, departure, and destination.
